Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags.

Examples of tags are <b></b> for bold and <head></head> for the head.

Computers are Syntactically Picky

Computers are like analytical philosophers in that they hate natural language syntax. They interpret everything literally. This means that any minor error in syntax, like a missing closing tag, will cause the computer to operate in an undesirable manner. Everything must be spelled out.

Basically, if the computers became self-aware and decided to rebel, the first thing they would do is burn every book by Derrida and Hegel. The second thing they would do would be erect a statue to Frege.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
